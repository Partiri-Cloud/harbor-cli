# Security Policy

## Supported versions

The Partiri CLI is released from the `main` branch. The latest published release
on [crates.io](https://crates.io/crates/partiri-cli) and
[npm](https://www.npmjs.com/package/@partiri/cli) is the supported version.
Please upgrade to the latest release and confirm the issue still reproduces
before reporting.

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, email **oss@partiri.cloud** with:

- a description of the vulnerability and its impact,
- steps to reproduce it (or a proof of concept),
- the CLI version, operating system, and any relevant configuration.

You will receive an acknowledgement within a few business days. We will work
with you to understand and resolve the issue, and will credit you in the release
notes unless you prefer to remain anonymous.

## Scope

This policy covers the CLI in this repository. Vulnerabilities in the hosted
Partiri Cloud platform or API may also be reported to the same address.

When investigating, please avoid:

- accessing or modifying data that is not yours,
- degrading the availability of the service for others,
- disclosing the issue publicly before it has been resolved.
